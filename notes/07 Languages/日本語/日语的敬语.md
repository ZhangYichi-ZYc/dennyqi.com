## 尊他语与自谦语的特殊形式

对于尊他语与自谦语有特殊形式的动词，一般优先使用其特殊形式，尊重程度也最高。

| 基本形     | 尊他语                    | 自谦语 |
| ---------- | ------------------------- | ------ |
| 見る       | ご覧になる                | 拝見する |
| 食べる     | 召し上がる                | いただく |
| 飲む       | 召し上がる                | いただく |
| 行く       | いらっしゃる/おいでになる | 参る/伺う |
| 来る       |いらっしゃる/おいでになる  | 参る/伺う |
| いる       | いらっしゃる/おいでになる| おる |
| する       | なさる | いたす |
| 言う       | おっしゃる | 申す |
| くれる     | くださる |        |
| ～ている/てある | ～ていらっしゃる | ～ておる |
| ～てくれる | ～てくださる |        |
| 知ってる   | ご存知です | 存じている |
| あげる | | 差し上げる |
| もらう | | いただく |
|            |                           |        |

其中，尊他语的いらっしゃる、なさる、おっしゃる、くださる在变为ます形时不遵守一类动词规则，而是把る变为います：いらっしゃいます、なさいます、おっしゃいます。
