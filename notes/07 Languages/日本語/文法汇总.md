## 句型

### 动词

#### 原型

| 含义                   | 文法                  | 例文                                                         |
| ---------------------- | --------------------- | ------------------------------------------------------------ |
| V＋こと＝N             | [V原]ことです         | 私の夢は先生になることです                                   |
| 有能力做（可能、允许） | [V原]ことが　できます | 私はピアノを弾くことができます                               |
| 之前                   | [V原]前に             | こちらへ来る前に、電話をかけてください；【ぼっち：下手なことを言う前に消えよう】 |
| 禁止 | [V原]な | タバコを吸うな【喜ぶな、喜ぶな】 |

#### 连用形

| 含义         | 文法                                 | 例文                                                     |
| ------------ | ------------------------------------ | -------------------------------------------------------- |
| 去/来 做某事 | [场所]へ　[V连用]に　行きます/来ます | 家へテレビを見に行きます                                 |
| 想做某事     | [V连用]たいです                      | このケーキを食べたいです                                 |
| 不想做某事   | [V连用]たくないです                  | もー、ひとりちゃん学校行きたくないからって、まだ噓ついて |
| …的方法      | [V连用]方                            | 話し方、作り方                                           |
| 提议         | [V连用]ませんか                      | 少し休みませんか                                         |
| 提议（口语） | [V连用]ましょう                      | 後藤さん、どこ行きたい？毎日思い出作りましょうね！       |
| 命令         | [V连用]なさい                        | 速く宿題を書きなさい                                     |
| 一边…一边…   | [V连用]+ながら                       | 小野さんはテレビを見ながら食事をしています               |

### 形容词

#### 形容词て形

| 含义                  | 文法                 | 例文                                       |
| --------------------- | -------------------- | ------------------------------------------ |
| 一类形容词的并列      | [Adjて]　Adjです     | このリンゴは大きくて、甘くて、美味しいです |
| 二类形容词/名词的并列 | [Adj/N]で　Adj/Nです | ここは便利で賑やかです                     |

#### 形容词た形

| 含义                                               | 文法                         | 例文                                                         |
| -------------------------------------------------- | ---------------------------- | ------------------------------------------------------------ |
| 列举形容词或名词（有的这样有的那样，有这个有那个） | [Adj/Nた]り　[Adj/Nた]りです | この公園は時間によって静かだったりにぎやかだったりです【ぼっち】：CD持って行って机に置いたり、バンドグッズもってアピールしたり… |

#### 副词性用法

一类形容词去い加く

二类形容词加に

| 含义           | 文法                    | 例文                                       |
| -------------- | ----------------------- | ------------------------------------------ |
| 修饰动词       | [Adv]＋V                | 静かにドアを閉めてください                 |
| 变得（现象）   | [Adv]＋なる             | 天気は暑くなりました；もう元気になりました |
| 使变得（人为） | [Adv]＋する；[N]に+する | テレビの音を大きくします；綺麗にしてるんだ |

#### 比较级，最高级

| 含义            | 文法                                               | 例文                                                         |
| --------------- | -------------------------------------------------- | ------------------------------------------------------------ |
| A比B更Adj       | AはBより（ずっと）ADJです                          | 王さんは小野さんより（ずっと）高いです【お酒とベースは　私の命より大事なものだから】 |
| 和B比还是A更Adj | Bより、Aのほうが　ADJです                          | 小野さんより、王さんのほうが高いです                         |
| A没有B来得Adj   | AはBほど　ADJ否定                                  | 小野さんは王さんほど高くないです                             |
| 还是Adj一点好   | [Adj1]/[Adj2+な]ほうがいいです                     | 部屋は広いほうがいいですね                                   |
| A里B是最Adj的   | Aの中でBが一番ADJです（场所、时间词后不加“の中”）  | スポーツの中でサッカーが一番面白いです                       |
| A和B哪个更Adj？ | AとBと　どちら（どっち）がADJですか？              | 日本語と英語と、どちらが難しいですか？                       |
| 哪个最Adj？     | Aの中で　どの名/いつ/どれ/誰/何　が一番ADJですか？ | 季節の中で　いつが一番好きですか？                           |
| 回答：都Adj     | どちらもADJです                                    | どちらも難しいです                                           |

### 助词

#### を

| 含义       | 文法        | 例文                       |
| ---------- | ----------- | -------------------------- |
| 做什么     | [对象]を　V | 私は毎日コーヒーを飲みます |
| 经过的场所 | N+を+V      | 公園を散歩する             |
| 离开的地点 | N+を+V      | 家を出る                   |
| 特例       | N+を+V      | 踊りを踊る                 |

#### が

| 含义             | 文法                         | 例文                                                   |
| ---------------- | ---------------------------- | ------------------------------------------------------ |
| 情感             | [主语]は[对象]が＋Adj        | 私は映画が好きです；私は犬が嫌いです                   |
| 能力             | [主语]は[对象]が＋Adj        | 私は日本語が分かりません；私はコンピューターができます |
| 想要             | [主语]は[对象]が＋欲しいです | 私は日本語が分かりません；私はコンピューターができます |
| 强调新信息       | [新]が[旧]です               | 課長は誰ですか？小野さんが課長です                     |
| 直接观察到的信息 | が                           | バスが来ました                                         |

#### に

| 含义     | 文法                             | 例文                                   |
| -------- | -------------------------------- | -------------------------------------- |
| 存在 有  | Aに　Bか　あります/います        | 机の上に　犬か　います                 |
| 存在 于  | Bは　Aに　あります/います        | リンゴは　箱の中に　あります           |
| 否定存在 | Aに　何もありません/誰もいません | 部屋に誰もいません                     |
| 时间     | [时间]に                         | 五時に会議があります                   |
| 动作对象 | [人]に　V                        | 私は明日小野さんに会います             |
| 目的地   | [目的地]へ/に　V                 | 私は日本に行きます                     |
| 次数     | [时间]に[次数]  V                | この花は２年に1度咲きます              |
| 附着点   | Nに V                            | 小野さんは電車に乗ります               |
| 用途     | Nに 使う                         | この写真は　パスポートの申請に使います |
| 对……来说…… | Nに Adj                           | この本は大人には易しいです。しかし、子供には難しいです |

#### で

| 含义     | 文法                      | 例文                                     |
| -------- | ------------------------- | ---------------------------------------- |
| 动作方式 | [交通工具]で（歩いて）　V | 私は地下鉄で家へ帰りました               |
| 动作场所 | [地点]で　V               | 昨日、私は王さんと図書館で本を読みました |
| 动作手段 | [手段]で　V               | 私は中国語で日記をつけます               |
| 表原因   | [N]で                     | 仕事で、小野さんと会っていたんです       |

#### と

| 含义                   | 文法          | 例文                                             |
| ---------------------- | ------------- | ------------------------------------------------ |
| 和…一起                | [人]と　V     | 毎日、私は王さんと一緒に学校へ行きます           |
| 多个主语               | AとBが        | 王さんと小野さんが結婚しました                   |
| 强调互动语感           | [人]と　V     | 私は小野さんとけんかしました                     |
| 恒常性状态，条件，习惯 | [V原/Vない]と | 食べ物を食べないと、人間は生きることができません |
| ……就好了 | [小句]といい | キタ：でもたくさん来てくれるといいわね |

#### から、まで

| 含义           | 文法                    | 例文                                 |
| -------------- | ----------------------- | ------------------------------------ |
| 从…到…（时间） | Aは　Tから　Tまで　V    | 私は　九時から　五時まで　勉強します |
| 从…到…（地点） | [起点]から[终点]まで　V | 私は　上海から日本まで　行きました   |
| 从…            | [来源地]から　V         | 私は中国から来ました                 |
